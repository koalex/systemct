### Cборка и запуск

*Установить зависимости*
```bash
yarn install
```

*Запуск сервера в production режиме*
```bash
yarn start
```

*Запуск сервера в production режиме (локально)*
```bash
yarn run start:local
```

*Запуск dev-сервера (режим разработки)*
```bash
yarn run dev
```

*Deploy*
```bash
yarn run deploy
```
